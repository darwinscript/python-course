## Server Inventory System

A lightweight Python CLI tool that manages a local server registry. It supports basic CRUD operations (create, read, update, delete) and persists data to a flat text file using a simple CSV-like format.

server_inventory.py

An in-memory inventory system built with dictionaries and lists. It allows adding, searching, updating, and deleting server records (hostname, IP, OS, role, services) through an interactive menu-driven interface, with automatic file persistence to **servers.txt**.
